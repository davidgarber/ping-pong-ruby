#Ping-Pong

<h2>by</h2>
David Garber

<h2>Description:</h2>
This app allows a user to count to a specified number and replaces numbers divisible by 3 with "Ping"
and numbers divisible by 5's with "Pong" and numbers divisible by both with "Ping-Pong."
